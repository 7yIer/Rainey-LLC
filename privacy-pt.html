Política de Privacidade

Última atualização: Maio 2025

Sua privacidade é importante para nós. Esta Política de Privacidade descreve como coletamos, usamos, divulgamos e protegemos suas informações quando você usa nossa aplicação.

1. Informações que Coletamos

  • Dados Pessoais: Nome, endereço de e-mail e outros detalhes que você fornece voluntariamente.
  • Dados de Uso: Tipo de dispositivo, sistema operacional, versão do aplicativo, padrões de uso e timestamps de acesso.

2. Como Usamos Suas Informações

  • Para fornecer e manter nosso serviço
  • Para melhorar e personalizar a experiência do usuário
  • Para comunicar atualizações e mudanças
  • Para fornecer suporte ao cliente
  • Para analisar padrões de uso e tendências
  • Para enviar conteúdo promocional (com consentimento)

3. Segurança dos Dados

Implementamos medidas de segurança padrão da indústria para proteger seus dados. No entanto, nenhum método de transmissão pela internet é 100% seguro. Nos esforçamos para proteger suas informações pessoais, mas não podemos garantir segurança absoluta.

4. Coleta e Uso de Dados Faciais

  • Imagens faciais e dados de escaneamento podem ser coletados para recursos de análise.
  • Os dados faciais são usados apenas para fornecer funcionalidade de análise e são principalmente armazenados localmente no seu dispositivo.
  • Os dados faciais podem ser temporariamente transmitidos para OpenAI/ChatGPT apenas para fins de análise e não são armazenados permanentemente em nossos servidores.
  • Não compartilhamos dados faciais com terceiros.
  • Os dados armazenados no seu dispositivo permanecem até você desinstalar o aplicativo.

5. Consentimento para Compartilhar Dados de Consumo com Apple

Ao usar nosso aplicativo e fazer compras no aplicativo, você consente em que compartilhemos dados sobre seu uso e consumo de conteúdo comprado com a Apple, como parte de nossos esforços para resolver solicitações de reembolso.

6. Mudanças nesta Política

Podemos atualizar esta política periodicamente. As mudanças serão publicadas aqui com uma data atualizada. Por favor, revise esta política regularmente.

7. Entre em Contato Conosco

Para perguntas sobre esta política, entre em contato conosco em raineyapps@gmail.com.
